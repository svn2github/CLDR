Locale Data Markup Language Errata

Summary: This document contains corrigenda for errata found in the 1.0 version of the Locale Data Markup Language specification (http://www.openi18n.org/spec/ldml/1.0/ldml-spec.html). These are errata that are too small to warrant generation of a new version of the specification, but which may be confusing to the reader.

This is a living document, and will be added to over time. In this header, only the date will reflect that changes have been made.
Date: 2003-08-03
Author: Mark Davis
  1. "Locale IDs"
    • the "currency=pre-euro" should be struck.  This is no longer a valid example.
  2. in  "XML Format, Common Elements : special"
    • %icu  should be %posix
  3. under "alias"
    •  there should be a closing  slash on the alias tag:   <alias .......... />
  4. Escaping Characters:
    • explain that '\u0000' is just a comparison to ICU format.    Compare to the &#x0000;  format.
  5. Page title (which shows up as the window title in the browser)
    • Should be "Locale Data Markup Language" instead of "Locale Data Interchange Format"
