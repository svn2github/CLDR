Overview

 

This document lists the differences between ICU vs. GCoC Toronto, ICU vs. Windows, ICU vs. Java and ICU vs. HPUX 11 locale data.

Locale data comparison for Solaris is in works.

 

XML format and DTD

 

Locale data has been compared only for a subset of data that has been mutually agreed upon conforming to DTD and reference XML files.

 

 

Color code key for differences

  • All entries that are equal are shown in the same color
  • Entries that are not equal to any other entry in the same row are shown in a different color

 

Collation XML

 

XML specification and DTD for Collation data can be found here.
