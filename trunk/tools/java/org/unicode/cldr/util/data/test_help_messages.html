This is a help-text file for use with the survey tool. The software that interprets this expects a particular format, so don't make arbitrary changes. You can add a new row, where the Key is usually a regex for a path, and you can put text to insert. The text to insert can actually be fairly arbitrary HTML.

Path Text to insert
.*/exemplarCharacters.* The standard exemplar characters are those used in customary writing ([a-z] for English; the auxiliary characters are used in foreign words found in typical magazines, newspapers, &c.; currency auxiliary characters are those used in currency symbols, like 'US$ 1,234'.
.*/numbers.*/pattern.* You need to use '.' for the decimal point and ',' for the thousands (grouping) separator, even if they are not used in your language. They are special characters that will be replaced by your real decimal point and thousands separator (see the number symbols for those).

The software that reads this table will search the first column (eg between <td> and </td>) and return the contents of the second column. I plan on adding a few variables also, for the current locale name, in particular.

WARNING

  • It uses a very dumb parser, so make sure that <tr>, </tr> are on separate lines!
  • The regex must match the whole path, so if it is an interior substring, remember to add .* on both ends.
