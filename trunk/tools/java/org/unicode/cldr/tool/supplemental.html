This set of tables shows the supplemental data contained in CLDR. This is data that is not language-based, but is required for use with the language-based data in CLDR. If you are vetting this data, please look over the tables to see how they are structured. Look for the countries and languages that you are familiar with, and see if the data is correct. If not, or if there is missing data, please file a bug. When you do so, please include references supporting the request.

Notes

Currencies

  1. The Country → Currency table shows which countries used which currencies, during which periods.
  2. The Currency Format Info table shows information organized by currency. The digits column shows the number of digits to use; if there is special rounding (such as for CH), that is in parentheses. The Countries column shows which countries the currency is — or has been — used in, officially.

Languages, Scripts, and Territories

  1. The Language → Territories and Territory → Language tables show modern official and major commercial languages for the territories in question. Cases where the language is used, but neither official nor a major commercial language are shown with *.
  2. The Language → Scripts and Script → Language do the same for scripts.
  3. The Not Represented tables list territories, scripts, and languages that are not represented in the above tables. These are for cross-checking. For example, "[Glag] Glagolitic" is found there because it is not in modern use. Other cases are there just because we don't yet have the data, for example, there is not yet language data for "[830] Channel Islands".

Calendars

  1. The Calendar data shows which calendars are in common use (other than the Gregorian calendar) in which territories.

Territory Containment

  1. The Territory Containment table shows the organization of territories and regions according to UN M.49, starting with the World. (CLDR supplements this table with the QO code for outlying areas that would not otherwise be included.) As the last column, the timezone IDs for that country are listed.
  2. The Windows-Tzid table shows the mapping from Windows timezone IDs to the standard TZIDs.

Metadata

  1. Aliases show how to map deprecated codes or aliases onto the ones that should be used to access CLDR data.
    • Note: Codes are shown in brackets before or after the English name, eg "Vanuatu [VU]"
  2. The Deprecated Items table shows CLDR items that are deprecated. Their usage remains valid, but they are strongly discouraged.

Source Data

The XML source data is in files in the following directory. Note that this file does not include the character fallback data, nor (most) metadata.

  • http://unicode.org/cldr/data/common/supplemental/
