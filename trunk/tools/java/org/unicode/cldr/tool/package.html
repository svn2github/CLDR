Tools

Provides tools for manipulating CLDR files.

Canonicalizing

Use CLDRModify -r to canonicalize the format.

Statistics

To generate statistics on CLDR, use: CountItems

-Dmethod=countItems -DSOURCE=C:\cvsdata\unicode\cldr\cldr_1_4\main\

-Dmethod=countItems -DSOURCE=C:\cvsdata\unicode\cldr\incoming\vetted\main\

It will generate something like:

Total Items 66,319
Total Resolved Items 1,025,077
Unique Paths 4,717
Unique Values 45,226
Unique Full Paths 9,301

Error Report

  • Run ConsoleCheckCldr with options
    • -f(.*) -p(.*) -t(.*) -n -s C:/cvsdata/unicode/cldr/incoming/vetted/main -zFINAL_TESTING -e -cminimal -g
  • Should take about 6 minutes.
  • Generates data in C:\cvsdata\unicode\cldr\dropbox\gen\errors\ (you can change this by adding to -g)
  • Once this is done, update with CVS.
